### JUJU异界

#### 版本

###### 最新版本

**1.0.0**

#### 安装依赖

运行 `npm install`，来安装所需的依赖模块。关于NPM的知识，请参见[nodejs](http://nodejs.org/);

#### 用Gulp打包

运行 `Gulp`，来对项目进行打包。关于Gulp的知识，请参见[Gulpjs](http://gulpjs.com/);

#### 目录说明

```
├── README.md     //描述项目
├── client        //开发
├── gulpfile.js   //gulp配置
├── package.json  //npm配置
├── build         //项目构建
├── examples      //demo
├── index.html    //项目入口
├── makefile.sh   //shell执行构建

```
